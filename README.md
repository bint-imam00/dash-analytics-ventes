# 📊 Sales Analytics Dashboard — Interactive Business Intelligence App

> **Built with Python · Dash · Plotly · Pandas**  
> An interactive dashboard for end-to-end sales analysis — from KPIs to regional heatmaps.

---

## 🚀 Live Demo

> Run locally in under 2 minutes — see instructions below.

---

## ✨ Features

- **📈 KPI Cards** — Real-time display of Total Revenue, Number of Orders, Average Basket, and Active Regions
- **📅 Monthly Trend Charts** — Evolution of sales over time with interactive line charts
- **🏆 Top Rankings** — Best performing states, cities, and product sub-categories
- **🗺️ Heatmaps** — Visual breakdown of sales intensity by geography and category
- **🔍 Dynamic Filters** — Filter by time period and region with instant chart updates
- **📱 Responsive UI** — Clean, professional layout that works on all screen sizes

## ⚡ Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/bint-imam00/dash-analytics-ventes.git
cd dash-analytics-ventes
```

### 2. Create virtual environment
```bash
python -m venv .venv

# Windows
.\.venv\Scripts\Activate.ps1

# Mac/Linux
source .venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Launch the app
```bash
python Aïcha_Mbaye_DIT_L3_Projet_Dash.py
```

### 5. Open in browser
```
http://127.0.0.1:8050
```
